# Day 02 - Introduction to Linux and Linux Distributions

## Objective

My goal for today was to understand what Linux is, learn what Linux distributions are, and become familiar with some of the most popular Linux distributions and their common use cases.

---

## What I Learned

- Linux is an operating system based on UNIX and is known for being free, open-source, secure, and efficient.
- Linux is widely used across servers, cloud environments, desktops, and many other types of devices.
- A Linux distribution is a complete operating system built around the Linux kernel together with system software, libraries, utilities, and applications.
- Different Linux distributions are designed for different purposes such as desktop use, servers, penetration testing, and lightweight systems.
- Each Linux distribution can come with its own package manager, desktop environment, default tools, and system design choices.
- Ubuntu is beginner-friendly and commonly used for desktops, servers, and cloud computing.
- Debian is known for stability and is widely used for servers.
- Kali Linux is focused on penetration testing, ethical hacking, and digital forensics.
- MX Linux is lightweight and designed for good performance on older hardware.
- Manjaro is user-friendly and gives access to newer software.
- Linux Mint is beginner-friendly and familiar for many Windows users.
- Solus focuses on simplicity and desktop performance.
- Fedora is popular with developers and is known for modern technologies.

## Key Takeaways

- Linux is the core operating system, while distributions are different packaged versions built around it.
- Different Linux distributions are made for different users and different purposes.
- Ubuntu and Linux Mint are easier starting points for beginners, while Debian is known for stability and Kali Linux is specialized for security work.
- Understanding Linux distributions helps me better understand the Linux ecosystem as a whole.
- Even on theory-based days, it is useful to keep practicing terminal commands so I stay comfortable in the environment.
