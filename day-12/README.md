# Day 12 - Basic Linux Commands

## Objective

My goal for today was to learn some basic Linux commands and understand how they are used for navigation, viewing directory contents, creating directories, moving between directories, and removing empty directories.

---

## What I Learned

- Linux commands are used to interact with the operating system through the terminal.
- Learning basic Linux commands helps beginners understand how Linux works and use it more effectively for daily tasks.
- The `ls` command is used to list files and directories in the current working directory.
- The `ls` command helps users quickly understand the contents of a directory.
- The `pwd` command is used to display the present working directory.
- The `pwd` command shows the full path of the current directory and helps users know their current location.
- The `mkdir` command is used to create new directories.
- The `mkdir` command helps organize files and directories by allowing users to create folders.
- The `cd` command is used to change the current working directory.
- The `cd` command allows movement between directories and can also return the user to the home directory when used without arguments.
- The `rmdir` command is used to remove empty directories.
- The `rmdir` command is helpful for cleaning up folders that do not contain files.

---

## Key Takeaways

- Basic Linux commands are essential for working comfortably in the terminal.
- Commands like `ls` and `pwd` help me understand where I am and what is inside a directory.
- Commands like `mkdir`, `cd`, and `rmdir` make it possible to create, enter, and remove directories as needed.
- These simple commands form part of the foundation for more advanced Linux usage.
- Becoming familiar with basic commands makes navigation and file management much easier.
