# Day 13 - More Basic Linux Commands

## Objective

My goal for today was to learn more basic Linux commands and understand how they are used for copying files, moving or renaming files, deleting files, viewing system information, locating files, creating files, creating links, displaying file contents, clearing the terminal, and checking running processes.

---

## What I Learned

- The `cp` command in Linux is used to copy files or directories from one location to another.
- The `cp` command is useful for making backups and duplicating files.
- The `mv` command is used to move or rename files and directories.
- The `mv` command can help reorganize files between directories or change file names.
- The `rm` command is used to permanently delete files.
- The `rm` command helps remove unwanted files and free disk space.
- The `uname` command is used to display system information.
- The `uname` command helps identify the operating system.
- The `locate` command is used to quickly find files using a database.
- The `locate` command is often faster than manual searching.
- The `touch` command is used to create empty files and update file timestamps.
- The `ln` command is used to create links between files and supports both hard and soft links.
- The `cat` command is used to display file contents and can also combine multiple files.
- The `clear` command is used to clean up the terminal screen by removing previous output.
- The `ps` command is used to display currently running processes and helps monitor system activity.

---

## Key Takeaways

- Linux has many simple but powerful commands for handling files and viewing system information.
- Commands like `cp`, `mv`, and `rm` are important for file management.
- Commands like `touch`, `ln`, and `cat` are useful for creating files, creating links, and viewing file contents.
- Commands like `uname`, `locate`, `clear`, and `ps` help with system awareness, searching, and terminal usage.
- Learning these commands gives me a stronger foundation for working more efficiently in Linux.
