# Day 10 - File System Navigation Commands in Linux

## Objective

My goal for today was to understand file system navigation commands in Linux and learn how to move through directories, view my current location, list files, and create new directories.

---

## What I Learned

- File system navigation commands in Linux are used to move through directories and explore the file structure using the command line.
- These commands help users locate, access, and manage files and folders efficiently within the Linux operating system.
- `pwd` means print working directory, and it shows the current location in the system.
- `ls` is used to list the files and directories in the current folder.
- `cd` is used to change from one directory to another.
- With `cd`, I can move into a nearby folder by using the folder name.
- I can also use `cd` with a full path to move directly to a specific folder.
- Navigation in Linux becomes easier when I understand the current directory, the target directory, and the path between them.
- `mkdir` means make directory, and it is used to create a new folder inside the file system.
- These basic navigation commands are important for moving around the Linux environment and working efficiently in the terminal.

---

## Key Takeaways

- Linux navigation commands make it possible to move around the file system confidently from the terminal.
- `pwd`, `ls`, and `cd` are some of the most important commands for understanding where I am and where I want to go.
- Using a full path with `cd` helps me reach a specific location directly.
- `mkdir` is useful when I need to create new folders for organizing my work.
- Learning these commands gives me a stronger practical foundation for using Linux every day.
